# 컴퓨터가 수를 표현하는 방법

## 1. 개요

컴퓨터의 메모리(RAM)는 매우 많은 0과 1을 저장할 수 있는 휘발성 기억장치다. 이때 0과 1을 저장할 수 있는 하나의 메모리 단위를 bit라고 하고, bit를 8개 묶은 것을 byte라고 한다.

이처럼 컴퓨터는 0과 1만 저장할 수 있기에 수나 데이터를 저장할 때도 이진법으로 바꿔서 저장한다.

## 2. 이진법이란?

이진법이란 0과 1로만 숫자를 표현하는 방법이다.
자리수가 올라갈 때마다 그 값은 이전 자리수의 2배가 된다.

6을 이진법으로 표현한다면 다음과 같이 풀어쓸 수 있다.

- 6
- 4 + 2
- (1 \* 2<sup>2</sup>) + (1 \* 2<sup>1</sup>) + (0 \* 2<sup>0</sup>)
- 110<sub>2</sub>

만약 크기가 4byte인 자바의 int 자료형에서 6을 표현한다면 다음과 같이 표현될 것이다.

```
00000000_00000000_00000000_00000110
```

## 3. 음수 표현법

음수를 이진법으로 표현하는 방법은 두가지가 있다.

1. 비트 부호 표현법
2. 2의 보수 표현법

### 3.1. 비트 부호 표현법

비트 부호 표현법은 가장 왼쪽 비트를 부호 비트로 사용하는 방법이다. 만약 수가 양수라면 비트 부호를 0으로, 음수일 경우 1로 표현한다.

다음은 부호 비트 표현법으로 -6을 표현한 것이다.

```
1000_0110
```

하지만 비트 부호 표현법은 후술할 문제로 사용되지 않고, 주로 2의 보수 표현법을 활용하여 정수를 표현한다.

다음은 비트 부호 표현법의 단점이다.

1. **두 가지의 0**: 비트 부호 표현법은 +0과 -0이 모두 가능하다. 이렇게 두 가지 0 표현이 존재하는 것은 불필요하게 메모리를 사용하며, 연산의 복잡성을 증가시킨다.
2. **연산의 복잡성**: 두 수를 연산할 때 두 수의 부호를 고려해야 해서 연산이 복잡하다.
3. **범위의 비효율성**: 같은 수의 비트를 사용할 경우, 비트 부호 표현법은 2의 보수 표현법에 비해 표현할 수 있는 정수의 범위가 제한적이다.

### 3.2. 2의 보수 표현법

2의 보수법은 비트 부호 표현법에 비해 연산을 간소화하고 더 많은 범위를 표현할 수 있다. 어떤 수 x에 값을 더했을 때 0이 되는 수를 2의 보수라고 한다.

2의 보수를 구하는 방법은 다음과 같다.

1. 모든 비트를 반전시킨다.
2. 그 결과에 1을 더한다.

다음은 -6을 2의 보수 표현법으로 구하는 과정이다.

0. 6은 4비트 이진수로 0110이다.
1. 비트 반전: 1001
2. 1 더하기: 1010
