# Class vs Object

자바를 처음 배울 때 클래스와 객체에 대해 배우지만 한번쯤 나의 언어로 정리해보면 좋을 것 같아 내용을 작성하게 되었다.

- **클래스**: 클래스란 객체의 틀 혹은 설계도로 객체가 가져야 할 상태와 행동이 정의되어 있다.
  - 예시: "자동차" 클래스는 속성으로는 "색상", "속도"를 가질 수 있고, 행동으로는 "주행"', "정차"등의 메서드를 가질 수 있다.
- **객체**: 객체는 클래스로부터 생성되어 메모리에 할당된 실체로 프로그램에서 사용할 수 있게 만든 것이다.
  - 예시: 자동차 클래스를 기반으로 "검정색 제네시스"와 같은 실체 객체를 만들 수 있다.
