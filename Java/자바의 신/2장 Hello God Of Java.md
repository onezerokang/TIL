# 2장 Hello God Of java

## 1. 컴파일 & 실행

JDK를 설치하면 자바를 컴파일하고 실행할 수 있게 된다.

JDK를 설치한 후 Hello World를 찍어보자.

먼저 에디터에서 코드를 작성하고, 콘솔에서 컴파일 및 실행을 하면 된다.
컴파일은 javac 명령어로, 실행은 java 명령을 사용한다.

- **컴파일**: '엮다'라는 의미로 내가 짠 코드를 바이너리 파일로 변환하고, 다른 파일들과 연결시키는 작업을 통해 컴퓨터가 실행할 수 있도록 하는 작업
- **바이너리 파일**: 컴퓨터가 이해하고 실행할 수 있는 이진 파일

컴파일을 마친 코드는 JVM에서 실행된다.

## 2. main 메소드

실행을 목적으로 하는 모든 자바 클래스는 main 메소드가 반드시 있어야 한다.
다음 클래스 같이 main 메소드가 없는 클래스를 실행하려고 하면 에러가 발생할 것이다.

```java
public class HelloGodOfJava {
}
```

```zsh
# 예외
Exception in thread "main" java.lang.NoSuchMethodError: main
```

메인 메소드는 반드시 다음과 같이 선언되어야 한다.

```java
public static void main(String[] args) {
}
```
