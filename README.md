# TIL

## TIL 작성 규칙

- 매일 매일 작성한다.
- 폴더의 이름은 대문자로 시작한다.
- 문서의 이름은 되도록 영어로 작성한다.
- 문서 내 제목은 되도록 한글로 작성한다.
- 문장의 간셜성을 위해 평어체를 사용한다.
