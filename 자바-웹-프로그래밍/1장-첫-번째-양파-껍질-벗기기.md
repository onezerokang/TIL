# 1장: 첫 번째 양파 껍질 벗기기

저자는 프로그래밍 학습 단계를 양파 껍질 벗기기에 비유한다.

한 번에 한 가지 지식을 깊게 학습하기보다는

다양한 지식을 얕게 학습한 후

일정 수준이 되면 다음 단계의 깊은 지식으로 서서히 깊이를 더해가라는 의미이다.

양파 껍질 학습법에는 첫 번째 양파 껍질 벗기기와 두 번째 양파 껍질 벗기기가 있다.

1. 첫 번째 양파 껍질 벗기기: 최소한의 웹 어플리케이션을 만들 수 있는 정도의 HTML, CSS, JS, Java, DB를 얕게 학습하고 프로그래밍이 무엇인지 이해하는 것. 이후 최소 1년간의 실무 경험을 쌓는다.
2. 두 번째 양파 껍질 벗기기: 웹 어플리케이션을 더 잘 만들기 위해 프레임워크를 직접 만들어보고, 빌드, 배포를 학습하고, 웹 브라우저와 웹 서버가 어떻게 통신하는지 이해하는 것.

## 두 번째 양파 껍질을 벗기기 위한 추천 도서

자바 웹 프로그래밍 Next Step은 두 번째 양파 껍질을 벗기기 위한 내용으로 구성되어 있다. 이 책 이외에 저자가 추천하는 도서는 다음과 같다.

- 이펙티브 자바: 코딩 관례와 효과적인 자바 사용법을 알 수 있음.
- 성공으로 이끄는 팀 개발 실천 기술: 빌드 배포 운영과 관련해 알아야 할 도구들에 대한 학습
- 그림으로 배우는 HTTP & Network
- 그림으로 공부하는 IT 인프라 구조
