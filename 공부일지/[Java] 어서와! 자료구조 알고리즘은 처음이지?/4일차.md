## Stack/Queue/Deque

- Stack: Last In First Out
  - Vector를 상속받는 클래스
- Queue: First In First Out
  - LinkedList가 구현하고 있는 인터페이스
- Deque: 양방향에서 데이터를 꺼낼 수 있는 자료구조
  - LinkedList가 구현하고 있는 인터페이스

```java
import java.util.Stack;
import java.util.Queue;
import java.util.Deque;
import java.util.LinkedList;

public class Main {
  public static void main(String[] args) {
    Stack<Integer> stack = new Stack<>();
    Queue<Integer> queue = new LinkedList<>();
    Deque<Integer> deque = new LinkedList<>();

    stack.push(1);
    stack.push(2);
    stack.pop();
    stack.peek();

    queue.offer(1);
    queue.offer(2);
    queue.poll();
    queue.peek();

    deque.offerFirst(1);
    deque.offerFirst(2);
    deque.offerLast(3);
    deque.pollFirst();
    deque.pollLast();
    deque.peekFirst();
    deque.peekLast();
  }
}
```

## Stack의 사용처

- Function Call
- Do/Undo
- Screen Stack

## Queue의 사용처

- Event Queue
- Job Scheduler
