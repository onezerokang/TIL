## Map

Map은 Array의 index로 빠르게 값을 찾을 수 있다는 장점과, List의 유연하다는 장점을 융합한 자료구조다.
Map은 내부에 Key를 인덱스로 사용하는 배열(bucket)과 배열의 요소에 연결된 List로 이루어져있다.

Map은 Key와 Value를 쌍으로 데이터를 저장하게 된다.

1. Key를 hash function을 통해 hash값으로 만든다. 이를 hashing이라 한다.
2. hash값을 내부 bucket의 index로 mapping한다.
3. Value를 index를 mapping한 요소에 연결된 List에 저장한다.
4. 만약 hash값이 충돌한다면 해당 List에 데이터를 저장한 후 링크로 연결한다.

주의: 커스텀 클래스를 Key로 사용할 경우 `hashCode()`를 오버라이딩 해야한다.

## java.util.Map

자바에는 Map 인터페이스가 있다. 다음은 Map 인터페이스를 구현하는 클래스다.

- java.util.Hashtable<K,V>
- java.util.HashMap<K,V>
- java.util.concurrent.ConcurrentHashMap<K,V>

아래는 Map 예제코드다.

```java
import java.util.Map;
import java.util.HashMap;

public class Main {
  public static void main(String[] args) {
    Map<String, Integer> phoneBook = new HashMap<>();

    phoneBook.put("Fire station", 119);
    phoneBook.put("Police office", 112);

    System.out.println(phoneBook.get("Fire station")); // 119
    System.out.println(phoneBook.getOrDefault("Call center", -1)); // -1

    if(phoneBook.isEmpty()) {
      System.out.println("Phone book is Empty");
    }

    phoneBook.remove("Fire station"); // 119
    phoneBook.remove("Police office", 113); // false

    System.out.println(phoneBook.values()); // [112]
    System.out.println(phoneBook.keySet()); // [Police offce]
  }
}
```
