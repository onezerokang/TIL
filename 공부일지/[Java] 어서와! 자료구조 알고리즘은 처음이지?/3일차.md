## Set

Set은 중복을 허용하지 않는 자료구조로 데이터를 입력하기 전 중복 검사를 하여 데이터의 저장여부를 결정한다.

<!-- 이미지 첨부   -->

Set은 데이터의 저장 순서를 보장하지 않는다.

주의: 커스텀 클래스를 HashSet에 저장할 경우 해당 클래스에서 hash값을 넘겨줘야 한다.

## java.util.Set

다음은 자바의 Set 인터페이스를 구현하는 대표적인 클래스다.

- HashSet
  - 내부에 Hashtable
- LinkedHashSet
  - 내부에 LinkedList
- TreeSet
  - 내부에 Tree

```java
import java.util.Set;
import java.util.HashSet;

public class Main {
  public static void main(String[] args) {
    Set<Integer> set = new HashSet<>();
    set.add(1);
    set.add(2);
    set.add(3);

    set.add(1);
    set.add(2);
    set.add(3);

    System.out.println(set); // [1, 2, 3]
  }
}
```
