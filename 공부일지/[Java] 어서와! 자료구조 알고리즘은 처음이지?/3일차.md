## Set

Set은 중복을 허용하지 않는 자료구조로 데이터를 입력하기 전 중복 검사를 하여 데이터의 저장여부를 결정한다.

<!-- 이미지 첨부   -->

Set은 데이터의 저장 순서를 보장하지 않는다.

주의: 커스텀 클래스를 HashSet에 저장할 경우 해당 클래스에서 hash값을 넘겨줘야 한다.

## java.util.Set

다음은 자바의 Set 인터페이스를 구현하는 대표적인 클래스다.

- HashSet
  - 내부적으로 HashMap을 사용한다.
- LinkedHashSet
  - 내부적으로 LinkedHashMap을 사용한다.
- TreeSet
  - 내부적으로 TreeHashMap을 사용한다.

```java
import java.util.Set;
import java.util.HashSet;

public class Main {
  public static void main(String[] args) {
    Set<Integer> set = new HashSet<>();
    set.add(1);
    set.add(2);
    set.add(3);

    set.add(1);
    set.add(2);
    set.add(3);

    System.out.println(set); // [1, 2, 3]
  }
}
```

Set을 이용하면 집합을 구할 수 있다.

- 합집합(집합 A의 원소와 집합 B의 원소를 모두 갖는 집합): `addAll()` 메서드를 이용한다
- 차집합(집합 A에는 속하지만 집합 B에는 속하지 않는 원소를 갖는 집합): `removeAll()` 메서드를 이용한다
- 교집합(집합 A, B에 대하여 양쪽에 속하는 요소 전체로 이루어지는 집합): `retainAll()` 메서드를 이용한다.

## List의 중복제거하기

### addAll 사용

```
List<String> list = Arrays.asList("A", "B", "C", "A", "B", "C");
Set<String> set = new HashSet<>(list);
System.out.println(set); // ["A", "B", "C"]
```

### steam 사용

```
List<String> list = Arrays.asList("A", "B", "C", "A", "B", "C");
List<String> list2 = list.stream().distinct().collect(Collectors.toList());
System.out.println(list2); // [A, B, C]
```
